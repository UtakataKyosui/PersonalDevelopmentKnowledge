# Design - 設計書

システムの設計書を管理するディレクトリです。

## ファイル構成

### 命名規則
- `[プロジェクト名]_architecture.md` - システム設計
- `[プロジェクト名]_database.md` - データベース設計
- `[プロジェクト名]_api.md` - API設計
- `[プロジェクト名]_ui.md` - UI/UX設計
